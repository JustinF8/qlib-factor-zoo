## 三、各层详细说明

### 第1层：YAML 配置入口

配置文件示例（位于 `qlib-main/qlib-main/examples/benchmarks/LightGBM/`）：

```yaml
# workflow_config_lightgbm_Alpha158.yaml
task:
    dataset:
        class: DatasetH
        module_path: qlib.data.dataset
        kwargs:
            handler:
                class: Alpha158                          # ← 指定使用哪个Handler
                module_path: qlib.contrib.data.handler    # ← Handler所在模块
                kwargs: *data_handler_config
```

只需修改 `class: Alpha158` 为 `Alpha101` / `GTJA191` / `Alpha360` / `TDXGS` 即可切换因子集。

### 第4层：Loader 表达式生成层

#### 4.1 `loader.py` — Alpha158DL / Alpha360DL

文件：`qlib-main/qlib-main/qlib/contrib/data/loader.py`

- **Alpha360DL**: 生成 360 个因子，来自 6 个基础字段 × 60 天回溯窗口
  - 字段: `CLOSE`, `OPEN`, `HIGH`, `LOW`, `VWAP`, `VOLUME`
  - 表达式: `Ref($close, -N)/$close`（价格类归一化）, `Ref($volume, -N)/$volume`（成交量归一化）
  - 命名: `CLOSE0` ~ `CLOSE59`, `OPEN0` ~ `OPEN59`, ...

- **Alpha158DL**: 通过 `config` 字典配置，生成约 158 个因子
  - `kbar`: K线特征（开盘/收盘/最高/最低）
  - `price`: 价格特征
  - `rolling`: 滚动统计特征（如 `Corr($close, Log($volume+1), 20)` 等）

#### 4.2 `loader_alpha101.py` — Alpha101DL

文件：`qlib-main/qlib-main/qlib/contrib/data/loader_alpha101.py`

- 生成 **101 个 Alpha 因子**（ALPHA001 ~ ALPHA101）
- 基于 Kakushadze (2015) "101 Formulaic Alphas"
- 使用特殊算子: `TsArgmax`, `TsArgmin`
- 每个因子是独立的数学表达式，例如：
  - `ALPHA001 = (Rank(TsArgMax(SignedPower((returns<0?stddev(returns,20):close),2),5))-0.5)`

#### 4.3 `loader_gtja191.py` — GTJA191DL

文件：`qlib-main/qlib-main/qlib/contrib/data/loader_gtja191.py`

- 生成 **191 个国泰君安 Alpha 因子**（GTJA001 ~ GTJA191）
- 基于国泰君安 2014 年研报
- 使用特殊算子: `SMA`, `TsArgmax`, `TsArgmin`, `Amount`
- 例如：`GTJA001 = (-1*Corr(Rank(Delta(Log($volume),1)),Rank(($close-$open)/$open),6))`

#### 4.4 `handler.py` 内联 — TDXGS

文件：`qlib-main/qlib-main/qlib/contrib/data/handler.py`（第434-692行）

- **无需外部 loader 文件**，表达式直接在 `get_feature_config()` 中硬编码生成
- 使用 44 个自定义算子（全部在 `custom_ops.py` 中定义）
- 循环生成 27 组、共 88 个通达信技术指标因子

---

### 第5层：custom_ops.py — 自定义算子层

文件：`qlib-main/qlib-main/qlib/contrib/data/custom_ops.py`

继承体系：
```
Rolling / ElemOperator  ← qlib/data/ops.py
    ├── TsArgmax, TsArgmin    (Alpha101/GTJA191 依赖)
    ├── SMA                    (GTJA191 依赖)
    ├── Amount                 (GTJA191 依赖)
    ├── MA, STD_TDX            (TDXGS 别名算子)
    └── ATR, RSI, WR, CCI, BOLL_UP/DN/MID, ...
        共44个 TDXGS 专用算子
```

每个算子实现 `_load_internal()` 方法，接收 pandas Series，返回计算结果。

| 维度 | Alpha101 | Alpha158 | Alpha360 | GTJA191 | TDXGS |
