## 一、因子库总览

| # | 类名         | 因子数量       | 因子分组           | 数据来源           | 特征类型     | 适用场景       |
| - | ------------ | -------------- | ------------------ | ------------------ | ------------ | -------------- |
| 1 | `Alpha360` | **360**  | 原始价量回溯 60 日 | Qlib 原生          | 日频价格/量  | 深度学习端到端 |
| 2 | `Alpha158` | **158**  | 价量统计特征       | Qlib 原生          | 日频价格/量  | 经典量化模型   |
| 3 | `Alpha101` | **101**  | WorldQuant Alpha   | Kakushadze 2015    | 日频价量截面 | 多因子选股     |

## 二、架构说明

### 2.1 文件结构

```
qlib/contrib/data/
├── handler.py              # 六个 DataHandler 类定义 (本文档核心)
├── loader.py               # Alpha158DL / Alpha360DL (Qlib 原生)
├── loader_alpha101.py      # Alpha101DL (101 因子表达式)
├── loader_gtja191.py       # GTJA191DL  (191 因子表达式)
├── custom_ops.py           # 自定义算子实现 (50+ 个算子类)
└── 六大因子库使用说明.md    # 本文档
```

### 2.2 类继承关系

```
QlibDataLoader                          DataHandlerLP
    │                                       │
    ├── Alpha360DL ──────────────► Alpha360 (使用 Alpha360DL)
    ├── Alpha158DL ──────────────► Alpha158 (使用 Alpha158DL)
    ├── Alpha101DL ──────────────► Alpha101 (使用 Alpha101DL)
    ├── GTJA191DL  ──────────────► GTJA191 (使用 GTJA191DL)
    └── (内联)    ──────────────► TDXGS   (内置 get_feature_config)
         (内联)    ──────────────► JQ110DataHandler (内置 get_feature_config)
                                        │
                                        └── JQ110DL (配套 Loader)
```

- **DataHandler**: 负责特征计算 + 数据预处理 + 标签生成
- **DataLoader**: 负责表达式计算 + 数据加载
- **custom_ops.py**: 提供 Qlib 表达式引擎不内置的算子

### 2.3 数据字段要求

所有因子库依赖 Qlib 日频数据，最小字段集：

| 字段   | 说明   | Qlib 变量   |
| ------ | ------ | ----------- |
| close  | 收盘价 | `$close`  |
| open   | 开盘价 | `$open`   |
| high   | 最高价 | `$high`   |
| low    | 最低价 | `$low`    |
| vwap   | 均价   | `$vwap`   |
| volume | 成交量 | `$volume` |

### 1. Alpha360 — 原始量价 360

**文件位置**: `handler.py` 第 112-154 行 (类定义)，`loader.py` (表达式)

**因子数量**: 360 个 (6 字段 × 60 日回溯)

**因子构成**:

| 字段   | 数量 | 命名格式                   | 说明                                |
| ------ | ---- | -------------------------- | ----------------------------------- |
| CLOSE  | 60   | `CLOSE0` ~ `CLOSE59`   | `Ref($close, d)/$close`           |
| OPEN   | 60   | `OPEN0` ~ `OPEN59`     | `Ref($open, d)/$close`            |
| HIGH   | 60   | `HIGH0` ~ `HIGH59`     | `Ref($high, d)/$close`            |
| LOW    | 60   | `LOW0` ~ `LOW59`       | `Ref($low, d)/$close`             |
| VWAP   | 60   | `VWAP0` ~ `VWAP59`     | `Ref($vwap, d)/$close`            |
| VOLUME | 60   | `VOLUME0` ~ `VOLUME59` | `Ref($volume, d)/($volume+1e-12)` |

**特点**:

- 所有价格字段用当日 close 归一化
- 成交量用当日 volume 归一化
- 不做任何统计加工，保留最原始的量价信息
- 适合深度学习模型端到端训练（如 LSTM、Transformer）

**默认标签**: `Ref($close, -2)/Ref($close, -1) - 1` (未来1日收益率)

**变体**: `Alpha360vwap` — 标签改用 vwap 计算

**使用代码**:

```python
from qlib.contrib.data.handler import Alpha360
handler = Alpha360(instruments="csi300", start_time="2010-01-01", end_time="2023-12-31")
```

### 3. Alpha101 — WorldQuant 101 公式化 Alpha

**文件位置**: `handler.py` 第 552-611 行 (类定义)，`loader_alpha101.py` (101个表达式)

**因子数量**: 101 个

**来源**: Kakushadze (2015), *"101 Formulaic Alphas"*, arXiv:1601.00991

**命名格式**: `ALPHA001` ~ `ALPHA101`

**特点**:

- 世界知名量化对冲基金 WorldQuant 发布的因子公式
- 表达式高度复杂，大量使用 Rank/CsRank/Corr/Cov/Delta 等截面算子
- 使用自定义算子 `TsArgmax`/`TsArgmin` 实现精确定位
- 所有因子为截面标准化后的 Z-Score 信号

**使用的自定义算子**:

| 算子               | 说明                 | 使用位置                     |
| ------------------ | -------------------- | ---------------------------- |
| `TsArgmax(X, N)` | 滚动窗口 argmax 取值 | Alpha #001, #057, #060, #098 |
| `TsArgmin(X, N)` | 滚动窗口 argmin 取值 | Alpha #124 等                |

**使用代码**:

```python
from qlib.contrib.data.handler import Alpha101
handler = Alpha101(instruments="csi500", start_time="2018-01-01", end_time="2023-12-31")
```

### 4. GTJA191 — 国泰君安 191 Alpha

**文件位置**: `handler.py` 第 614-676 行 (类定义)，`loader_gtja191.py` (191个表达式)

**因子数量**: 191 个

**来源**: 国泰君安证券研究所 2014 年 Alpha 因子研报

**命名格式**: `GTJA001` ~ `GTJA191`

**特点**:

- A股市场经典因子集，广泛用于券商研报和量化选股
- 大量使用 `SMA(X, N, M)` 算子（中国式递归移动平均）
- 使用 `TsArgmax`/`TsArgmin` 实现方向性因子
- 使用 `Amount()` 算子获取成交额（底层用 `$vwap*$volume` 代理）

**使用的自定义算子**:

| 算子               | 说明                       | 使用位置       |
| ------------------ | -------------------------- | -------------- |
| `SMA(X, N, M)`   | 递归移动平均（通达信口径） | 全集合大量使用 |
| `TsArgmax(X, N)` | 滚动窗口 argmax 取值       | 多处           |
| `TsArgmin(X, N)` | 滚动窗口 argmin 取值       | 多处           |
| `Amount()`       | 成交额访问器               | 成交额相关因子 |

**使用代码**:

```python
from qlib.contrib.data.handler import GTJA191
handler = GTJA191(instruments="csi300", start_time="2015-01-01", end_time="2023-12-31")
```

### 6. JQ110 — 聚宽 110+ 因子

**文件位置**: `handler.py` 第 158-484 行 (JQ110DataHandler + JQ110DL)

**因子数量**: ~113 个

**来源**: 基于聚宽（JoinQuant）用户策略脚本的 `calculate_all_factors()` 函数

**命名格式**: `JQ110_<指标名>_<参数>`

| ------------------- | ------------------------------------------------------------------------------------------------------------------- | ------------ | -------- |
| **MOMENTUM**  | ROC(5)/BIAS(4)/Aroon(2)/BBI(2)/CCI(4)/CR/MASS/TRIX(2)/Price1M~1Y(3)/Rank/BullBear(2)/VPT(4)/Volume1M/PLRC(3)        | **38** | 动量类   |
| **EMOTION**   | VOL(6)/DAVOL(3)/turnover_vol/TVMA(4)/VEMA(4)/VSTD(2)/ARBR(3)/ATR(2)/PSY/VMACD(3)/VOSC/VR/VROC(2)/WVAD(2)/money_flow | **36** | 情绪量能 |
| **TECHNICAL** | EMA(1)/EMAC(5)/MAC(5)/MACDC/BOLL(2)/MFI                                                                             | **17** | 技术指标 |
| **RISK**      | Variance(3)/SharpeRatio(3)/Skewness(3)/Kurtosis(3)                                                                  | **12** | 风险统计 |
| **STYLE**     | daily_std/hist_sigma/residual_vol/cumulative_range/momentum/liquidity/share_turnover(3)/beta                        | **10** | 风格     |

**特点**:

- 与 TDXGS 互补：TDXGS 偏重通达信原生指标，JQ110 偏重聚宽策略因子
- 包含 RISK 分组（Variance/Skewness/Kurtosis/SharpeRatio），其他因子库不含
- 包含 STYLE 分组（beta/换手率/流动性），其他因子库不含
- 使用自定义算子：`AroonUp/Down`、`BullPower/BearPower`、`VPT`、`WVAD`、`VOSC`、`Variance/Skewness/Kurtosis/SharpeRatio`、`PriceRank`、`CumulativeRange`、`MoneyFlow`

**默认标签**: `Ref($close, -5)/$close - 1` (未来5日收益率)

**使用代码**:


## 四、使用示例

### 4.1 单独使用一个因子库

```python
import qlib
from qlib.contrib.data.handler import Alpha158, TDXGS

# 初始化 Qlib
qlib.init(provider_uri="~/.qlib/qlib_data/cn_data")

# 使用 Alpha158
handler = Alpha158(
    instruments="csi300",
    start_time="2018-01-01",
    end_time="2023-12-31",
    freq="day",
)

# 获取特征和标签 DataFrame
df = handler.fetch()
print(df.columns.tolist())  # 158 个特征列 + 标签列
```

### 4.2 模型训练流水线

```python
import qlib
from qlib.contrib.data.handler import Alpha158
from qlib.contrib.model.pytorch_alstm import ALSTM
from qlib.contrib.strategy import TopkDropoutStrategy
from qlib.backtest import backtest, executor

# 数据处理
handler = Alpha158(instruments="csi500", start_time="2015-01-01", end_time="2023-12-31")

# 数据集
from qlib.data.dataset import DatasetH
dataset = DatasetH(handler=handler, segments={
    "train": ("2015-01-01", "2020-12-31"),
    "valid": ("2021-01-01", "2022-12-31"),
    "test": ("2023-01-01", "2023-12-31"),
})

# 模型
model = ALSTM(d_feat=158, hidden_size=64, num_layers=2, dropout=0.0, n_epochs=100)
model.fit(dataset)

# 预测
pred = model.predict(dataset)
```

### 4.3 混合使用多个因子库

如需将多个因子库的特征合并使用，需要自定义 Handler：

```python
from qlib.contrib.data.handler import Alpha158, TDXGS, JQ110DataHandler
from qlib.data.dataset.handler import DataHandlerLP

class MixedHandler(DataHandlerLP):
    def __init__(self, instruments="csi300", **kwargs):
        # 分别获取各因子库的特征
        alpha158_fields, alpha158_names = Alpha158().get_feature_config()
        tdxgs_fields, tdxgs_names = TDXGS().get_feature_config()
        jq110_fields, jq110_names = JQ110DataHandler.get_feature_config()

        # 合并特征
        all_fields = alpha158_fields + tdxgs_fields + jq110_fields
        all_names = alpha158_names + tdxgs_names + jq110_names

        super().__init__(
            instruments=instruments,
            data_loader={
                "class": "QlibDataLoader",
                "kwargs": {
                    "config": {
                        "feature": (all_fields, all_names),
                        "label": ["Ref($close, -2)/Ref($close, -1) - 1"],
                    },
                    "freq": "day",
                },
            },
            **kwargs,
        )
```

### 4.4 各因子库快速对比

| 场景                               | 推荐因子库                           |
| ---------------------------------- | ------------------------------------ |
| 深度学习端到端（LSTM/Transformer） | `Alpha360`                         |

| 国际量化因子研究                   | `Alpha101`                         |
| A股传统多因子选股                  | `GTJA191`                          |

## 五、自定义算子总表

所有自定义算子定义在 `custom_ops.py`，按分组列出：

### 5.1 基础算子（Alpha101/GTJA191 依赖）

| 算子         | 签名               | 说明                             |
| ------------ | ------------------ | -------------------------------- |
| `TsArgmax` | `TsArgmax(X, N)` | 滚动窗口最大值位置取值           |
| `TsArgmin` | `TsArgmin(X, N)` | 滚动窗口最小值位置取值           |
| `SMA`      | `SMA(X, N, M)`   | 中国式递归移动平均（通达信口径） |
| `Amount`   | `Amount()`       | 成交额（`$vwap*$volume` 代理） |

### 5.2 通达信技术指标（TDXGS 依赖）

| 算子           | 签名                                     | 说明                            |
| -------------- | ---------------------------------------- | ------------------------------- |
| `ATR`        | `ATR($close, $high, $low, N)`          | 平均真实波幅                    |

## 七、注意事项

### 7.1 数据要求

- 所有因子库依赖 Qlib 日频数据的 6 个基础字段（close/open/high/low/vwap/volume）
- 使用前需先通过 `qlib.init(provider_uri="...")` 初始化数据源
- 数据回溯长度需大于最大窗口参数（如 JQ110 需要 250+ 日，Alpha360 需要 60+ 日）

### 7.2 因子重叠

- `Alpha158` 与 `TDXGS` 有部分重叠（如 ROC/MA/STD 等基本统计量）
- `TDXGS` 与 `JQ110` 在技术指标上有重叠（如 ATR/CCI/PSY/VR 等）
- 混合使用时建议做去重或相关性分析

### 7.3 性能考虑

- `Alpha101` 和 `GTJA191` 包含大量截面算子（CsRank/Corr），计算量较大
- `Alpha360` 因子数量最多（360），内存占用最大
- `TDXGS` 使用较多自定义算子，部分算子涉及 `rolling().apply()`，性能略低于 Qlib 内置算子
- 建议先用 `Alpha158` 作为 baseline，再逐步增加其他因子库

### 7.4 标签配置

| 因子库                                   | 默认标签公式                            | 含义                    |
| ---------------------------------------- | --------------------------------------- | ----------------------- |
| Alpha360/Alpha158/Alpha101/GTJA191/TDXGS | `Ref($close, -2)/Ref($close, -1) - 1` | 未来1日收益率           |
| JQ110                                    | `Ref($close, -5)/$close - 1`          | 未来5日收益率           |
| Alpha360vwap/Alpha158vwap                | `Ref($vwap, -2)/Ref($vwap, -1) - 1`   | 基于vwap的未来1日收益率 |

### 7.5 已知限制

- 所有因子库仅支持**日频数据**，不包含分钟级因子
- **不包含财务数据**因子（如 PE/PB/ROE/利润增长率等），如需基本面因子需要额外引入财务数据源
- Beta 因子使用 `Slope` 近似，非标准市场 Beta（需要市场基准收益率序列）
- `Amount()` 算子使用 `$vwap*$volume` 代理成交额，与真实成交额可能存在微小偏差
